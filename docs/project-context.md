- 리포지토리 초기화 계획 이후 2년 이상의 지연
- 최신 React·Next.js 생태계와 하네스 엔지니어링 경험 필요
- GEO·AEO·접근성·성능 최적화의 실전 경험 필요
- 모바일 우선 레이아웃과 새로운 디자인 시스템 적용 필요
