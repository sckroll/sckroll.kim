# Phase 07 — 실제 콘텐츠 전환과 공개 배포

## 상태

- 상태: 대기
- 진입 조건: Phase 06 완료
- 목적: 더미 콘텐츠 제거와 실제 콘텐츠 기반 공개 운영 시작
- 외부 변경: 사용자 승인 이후의 배포·도메인·검색 도구 연결

## 실제 콘텐츠 전환

- [ ] [`content-track.md`](content-track.md) 완료 확인
- [ ] 공개 프로젝트 최소 2개 입력
- [ ] 공개 포스트 최소 1개 입력
- [ ] 소개·이력서 실제 정보 입력
- [ ] 미션·비전·핵심 가치 표현 최종 편집
- [ ] 실제 이미지·영상·대체 텍스트 입력
- [ ] NDA·저작권·보안 최종 확인
- [ ] 샘플 표기와 더미 콘텐츠 전부 제거
- [ ] fixture 전용 경로의 production 제외 확인
